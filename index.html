Electro Store

August 2020

This is a E-commerce web store that sell mobile phones. The user choose their favourite product/products and then these products will be added to a shopping cart that represents an array of objects where each object represents the product properly. After choosing the products the user needs to complete a form using their data to finish the purchase. If the user miss some required data, a realtime message will be rendered to them on the form bottom, then if data is correct after Send Command click event button the client makes an HTTP POST request to the server to send the user data and their selected products from the cart.

The app was created using ReactJs framework, HTML5 and CSS3 for the client side, also for the backend side was used NodeJs + ExpressJs and PostgreSQL as database. General tools used: NPM package manager, Git, Visual Studio Code and Postman.

Hobby Explorer

January 2021

This project represents a web application that helps people find each other by their mutual hobbies and let them communicate through messages to meet and practice that hobbies/passions. The user has also a section where he can see his general profile informations and change his username or location. Being a mini social media application the user needs a profile picture to create the account. This application was created using ReactJs framework, CSS3 and some React Bootstrap elements for the UI styling. For the API backend service was used NodeJs with ExpressJs for the server and PostgreSQL and Firebase service for database.

Tools used: Visual Studio Code, Git, Postman and NPM package manager.
